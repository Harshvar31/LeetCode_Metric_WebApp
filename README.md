#  LeetCode Metrics WebApp

## About The Project
LeetCode Metrics WebApp is a simple and interactive dashboard that fetches and displays a user's LeetCode performance statistics.  
It provides a clean UI to visualize coding progress and problem-solving activity.

This project was built to practice API handling, DOM manipulation, and frontend development concepts.

---

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LeetCode API  

---

##  Features
-  Search LeetCode user by username  
-  Display total solved problems  
-  Show difficulty-wise distribution  
-  Clean and responsive UI  
-  Fast and lightweight
